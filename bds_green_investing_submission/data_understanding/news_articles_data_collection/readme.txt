This folder includes all data and scripts for news articles

1. Run newsarticles.py to get [company].csv
2. [company]_new.csv is the above but manually fit the columns for import into pandas.
3. Then run cleancv.py. This uses [company]_new.csv to generate [company]_cleaned.csv
4. Finally run cleandata.py. This uses [company]_cleaned.csv to generate [company]_datacleaned.csv
